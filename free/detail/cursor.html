Cursor 的缺点

虽然 Cursor 功能强大，但也存在一些局限性需要了解

专业版价格较高

免费版有明显的功能限制，专业版订阅费用对于个人开发者来说较高，长期使用成本不容忽视。

依赖网络连接

大部分AI功能需要网络连接才能使用，在没有网络的环境下，Cursor 与普通编辑器相比优势不大。

扩展生态有限

与 VS Code 等成熟编辑器相比，Cursor 的扩展生态系统还不够完善，可用插件数量较少。
