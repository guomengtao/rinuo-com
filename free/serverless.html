免费 Serverless 后端 API 服务推荐

AWS Lambda + API Gateway

访问官网
免费额度
  • 每月 1M 请求
  • 400K GB-秒 / 3.2M 秒 计算时间
限制
  • 需配置 API Gateway，稍有学习成本
适用场景
  • 轻量实时后端 API 服务

Cloudflare Workers

访问官网
免费额度
  • 每天 100,000 请求
  • 10GB 出站带宽
限制
  • 请求量和存储有限
适用场景
  • 边缘部署 API、轻逻辑处理

Netlify Functions

访问官网
免费额度
  • 每月 125,000 函数调用
限制
  • 仅支持静态 + 函数组合部署
适用场景
  • JAMstack 架构 API 服务

Vercel Serverless Functions

访问官网
免费特点
  • 支持 Node.js、Go 等
  • 自动生成 API URL
限制
  • 冷启动时间较高（约 8 秒）
适用场景
  • 与前端项目一起快速开发

Google Cloud Functions

访问官网
免费额度
  • 每月 2M 调用
  • 400K GB-秒
  • 5GB 出站流量
限制
  • 需注册 GCP，额度耗尽后收费
适用场景
  • 事件驱动或 HTTP API 服务

Appwrite（开源 BaaS）

访问官网
免费特点
  • 自托管且开源，包含函数、Auth、DB
限制
  • 需自建服务器，运维成本较高
适用场景
  • 小型项目或完全掌控环境需求

NHost

访问官网
免费特点
  • 提供 Postgres、GraphQL API、Auth、Storage、函数
限制
  • 1GB DB 和 5GB 出站流量免费额度
适用场景
  • 快速生成后端 MVP

整理于 2025-08-29。内容参考自 AWS 官方, Google Cloud, Netlify, Vercel, 及多篇开发者推荐文章
