  • AWS Lambda + API Gateway — 稳定成熟，适合构建轻量 API。

  • Vercel Serverless Functions — 与前端项目部署一体化，易测试。

  • NHost — 集成 Postgres、Auth、GraphQL API 的轻量后端。
